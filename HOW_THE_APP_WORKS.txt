High-Level Overview

What is LeaseWise?

LeaseWise is an AI-powered lease analysis tool that helps tenants understand their rental agreements. It uses **Retrieval Augmented Generation (RAG)** to ensure all information is grounded in the actual lease document, preventing AI hallucinations.

Key Design Principles

1. **RAG-First Approach
Every analysis is grounded in actual lease text. No hallucinations.

2. **Multi-Query Strategy
Complex features use multiple targeted queries to gather comprehensive information.

3. **Source Attribution
Every fact includes its page number for verification.

4. **Parallel Processing
Multiple analyses run simultaneously for speed.

5. **Verification Layers
Legal information passes through multiple checks before display.

6. **Chunk Reuse
Embeddings are created once and reused for all features, saving time and cost.
